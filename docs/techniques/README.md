# Plotter Art Techniques Library

A knowledge base of drawing/art techniques relevant to pen plotter generative art, organized for inspiring new hatch3d compositions.

## How This Is Organized

Each technique file covers: what it is, the core algorithm, how it maps to plotter output, and composition ideas for hatch3d.
